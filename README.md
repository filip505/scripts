# Scripts

Scripts that makes my life easier

## Historys

Save last entered command from terminal

```bash
sh ./historys s
```

List all saved commands

```bash
sh ./historys l
```

## StartDrive

Encript HDD and start file sharing

```bash
sh ./startDrive
```

## EndDrive

Decrypt HDD and stop file sharing

```bash
sh ./endDrive
```
